Blokada blocks less ads.

If you suspect that Blokada blocks less advertisements than earlier, please refresh your lists. To do it, tap on Blacklist, then in the top right corner on the equaliser, choose Reflesh filters and tap on Continue. Usually it happens because new servers are added to the existing ones which are not (yet) identified and added to the blacklists, or the new list is not downloaded to your phone, therefore they are not (yet) blocked.

Alternatively you can enable more host lists or add more restrictive filters in the Blacklist menu.

An app doesn't work as expected.

If you can see that an app doesn't work as expected, doesn't load its content or something similar misbehaviour is experienced, please go to the main screen, choose Whitelist, at the bottom of the screen tap on the + sign, choose App and type its name. Tap on Save at the end. With this, Blokada won't filter the URLs in this app.

I want to block a given website.

You can block individual website which isn't on a host list. To do it, go to the Blacklist on the main page, tap on the + sign at the bottom of the screen, tap on Host and type in the hostname if the website, without the http and www part (for example: myexample.com).

I cannot start Blokada after I installed it.

There are two common causes for such a situation. First, please check if you have an active VPN connection and if thats Always-on VPN option is enabled. To do it, go to the Settings of your phone, search for VPN in the network section, tap on the gear next to the existing VPN. Disable it, disconnect from the active VPN connection and try to restart Blokada.

If you don’t have other VPN connection, check in Blokada if Online only is enabled, and if yes, check if you are connected to the internet either with wifi or with mobile data. Blokada won’t start if the Online only is enabled and you don’t have an internet connection.

If it doesn’t help, restart your phone, it also helps sometimes.

Blokada stops to function/work after a while.

As a first step, turn on Keep alive in Blokada. In this case, Blokada will try to stay online, but please note, your system still can effect this behaviour. If you set it, and you still see that Blokada is stopped time after time, check your system’s settings and turn off every battery- and memory-optimisation for Blokada. Unfortunately it is different in every system, therefore we cannot say where to do it. Please check your phone’s settings. Other source of such behaviour can be that you are using a memory cleaner. Please set Blokada on whitelist or exclude list. If non of this steps helped in your case, please report a bug (see below how to do it).

¿Cómo funciona Blokada?

Blokada is a thin layer between the network interface (wifi or mobile) of your phone and the application/s you use. To be able to filter every outgoing traffic, it generates a VPN connection and simply blocks any requests towards the hosts defined in the blacklist filters. This blocking is done with a redirection, the IP address of the website which serves the advertisement will be resolved by the phone to a non-routable meta-address, for 0.0.0.0, where no web service is running, therefore nothing will be shown. With this, Blokada saves both your data plan and battery life. The filters are updated automatically and contain hosts known for serving ads, tracking, spam, malware, and other unwanted or dangerous content. You can configure Blokada with your own custom filters and whitelists to ensure the service which you would like to use isn’t blocked and you don’t experience any interruption.

¿Puedo usar otra VPN mientras que Blokada esté activado?

Unfortunately no, it is a system limitation; you can have only one active VPN connection at a time. If you join to an other service, Blokada will exit.

¿Cuáles son las funciones de Blokada?

As you surely saw it, Blokada has some options on the main screen which are not detailed at the first start. Please let us do it now. You can see the following:

  • Blacklist: this is the lists of host files containing the servers to block. Please note that using multiple lists can have effect on network and system speed.
  • Whitelist: the whitelist is the exception list, here you can define hosts which should not be blocked or apps where Blokada won’t filter URLs. Blokada is installed with several whitelisted apps by default.
  • Mantener activo: esta opción es para asegurar que Blokada no sea interrumpido o detenido por el sistema. Ten en cuenta que tu sistema puede sobreescribir este ajuste. Asegúrate de que tus ajustes dejen a Blokada ejecutarse en segundo plano.
  • Notifications: when enabled, you will be notified when an ad is blocked. It is on by default, but if you get a lot of notifications, turn it off.
  • Sólo en línea: cuando está habilitado, Blokada se detendrá si no hay una conexión activa, y se volverá a iniciar cuando vuelva la conexión. Si está desactivado, Blokada siempre se ejecutará en segundo plano (incluso si no hay ninguna red disponible). Ten en cuenta que esta opción puede no funcionar en todos los dispositivos.
  • Start on boot: when enabled, Blokada will start after the system boots up. Please note there are system level configurations for this option, therefore it cannot be guaranteed Blokada can start on boot. In this case, please check your phone’s settings. The default configuration is enabled, if you want to start Blokada manually, disable this option.

Not configurable things:

  • Up to date: if you see it, you are running the most recent version of Blokada. Keep the app up-to-date because it has less bugs and more features!
  • # ads blocked: the number of unwanted content that Blokada blocked since it is started for the first time
  • # hosts: number of servers that serves advertisements, malware and other unwanted content

¿Qué puedo hacer sus sigo viendo anuncios?

Algunos anuncios seguirán llegando, incluso, con bloqueador de anuncios en Android. En particular, los anuncios de YouTube o algunos anuncios popup a pantalla completa son difíciles de bloquear. Si piensas que están llegando más anuncios que antes, prueba estos pasos:

  • Comprueba si el icono de la llave VPN está presente en la barra de estado de la parte superior. Si no, arranca Blokada.
  • Abre Blokada y comprueba si está activado y que no reporta ningún error.
  • Comprueba si el contador de hosts no es igual a cero. Si es así, selecciona algunos filtros de listas negras para bloquear los anuncios. Si el contador sigue siendo cero, informa del fallo.
  • Asegúrate de que no tienes demasiadas aplicaciones en la lista blanca (tal vez por eso es que estás viendo anuncios).
  • Try resetting your filters (go to the Blacklist screen and choose Restore Defaults from the Quick Actions menu at the top).
  • If you see ads in your browser, deactivate any kind of traffic compression or optimisation in the browser's settings.
  • Desactiva Blokada, ciérralo (sacándolo de la pantalla Recientes) e inícialo de nuevo.
  • Si has cambiado recientemente la configuración (filtros, lista blanca, etc.), recuerda que lleva tiempo antes de que la nueva configuración empiece a funcionar. Dale un poco de tiempo.
  • Borra los datos de Blokada (en la configuración del sistema), o reinstálalo.
  • Reinicia el dispositivo.
  • Your device may be unsupported. Please check Devices Database.
  • Si ninguno de esos pasos te ha servido, por favor informa de un fallo (ver abajo).

No puedo instalar una actualización. ¿Qué puedo hacer?

First of all, please check if the Android version you have is 5 or newer. Android 4 and older versions are not supported in Blokada 3.0 and newer versions. To do it, go to the Settings of your phone, scroll to About phone and search for Android version

If your phone runs Andriod 5 or newer, please restart your device. It sounds strange, but sometimes it helps. If that didn't solve the issue, uninstall Blokada and install it again. It can be, that your device is unsupported. Please check Devices Database.

If none of those steps helped, please join to our Telegram chat on go.blokada.org/chat to get help, or report a bug (see below).

¿Puedo ocultar la notificación de la VPN?

Desafortunadamente no. Algunos dispositivos pueden mostrar notificaciones permanentes mientras la VPN está activa, y no es posible ocultarla. Si puedes, intenta actualizar el dispositivo, ya que las nuevas versiones de Android no tienen este problema.

¿Cómo reporto un problema o una idea?

If you are experiencing problems, please join to our Telegram chat at go.blokada.org/chat, where our community is happy to help you. Alternatively, just choose the Feedback option from the menu in top right corner and follow the instructions. Please provide as much information as you can and steps to reproduce the same issue - it helps greatly in fixing your problem! If you are a pro user, you can go directly to the issue tracker at GitHub.

¿Puede Blokada acceder a mi información personal?

No. Blokada does not read or tamper with your private information. Blokada only monitors which hosts your device is making requests towards and blocks the ones matching the filters. Every time Blokada blocks anything, you will see a notification (which you can turn off). Blokada has no access to your encrypted traffic.

¿Blokada gasta mucha batería?

No. Depending on your settings the battery impact is minimal, and on most new devices, especially on Marshmallow (Android 6) or newer, you will not notice any battery drain. However, some unlucky devices may experience increased battery drain. Refer to the Devices Database for details.

¿Blokada ralentizará mi dispositivo?

No. Depending on your settings Blokada can be quite efficient. In fact, both your device's performance and battery life may visibly improve, because Blokada saves your device from downloading and processing content you don't need or care about. However, users with some older devices may experience performance drop. Refer to the Devices Database for details.

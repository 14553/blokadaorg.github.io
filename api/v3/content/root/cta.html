March 2018: Blokada Branding Vote (tap to vote)

This month we are voting to see how we like the name "Blokada". You can vote that we keep the name, use one of the existing suggestions, or propose your own name that you think fits the project better. Your suggestion will be added to the list.

By the end of the month, based on voting results, we'll decide whether we do the rebranding or stick with the current name.1
